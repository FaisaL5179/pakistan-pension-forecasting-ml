# Forecasting Pakistan's Pension System Using Machine Learning Techniques

Final Year Project developed for the BS Computational Finance program at NED University of Engineering & Technology.

## Project Overview

This project focuses on forecasting pension obligations using Machine Learning and Time Series models on historical pension-related financial data from major Pakistani institutions and PSX-listed organizations.

## Models Used

- Linear Regression
- Random Forest
- ARIMA
- XGBoost
- CatBoost

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost
- CatBoost
- Statsmodels
- Matplotlib
- Jupyter Notebook

## Dataset

The dataset consists of pension-related financial data collected from annual reports of major Pakistani organizations.

## Evaluation Metrics

- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- Mean Absolute Error (MAE)
- R² Score

## Repository Contents

- FYP_Report.pdf

## Results

The developed models were used to forecast pension obligations for 2025 and 2026 and compare forecasting performance across different machine learning approaches.
